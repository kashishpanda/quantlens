# QuantLens

Institutional Quant Research Platform with ML Intelligence

## Vision
QuantLens is a production-style quantitative research platform designed to simulate infrastructure used by institutional finance teams.

The platform will ingest market data, clean and validate datasets, compute quantitative analytics, support strategy backtesting, and apply machine learning for anomaly detection and market intelligence.

## Tech Stack
- Python
- FastAPI
- PostgreSQL
- Pandas
- Streamlit
- Docker
- scikit-learn
- AWS (later)
